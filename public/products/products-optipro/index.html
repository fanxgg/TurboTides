OASIS

OASIS

Download PDF

Next-generation optimization software

OASIS delivers next-generation optimization capabilities that revolutionize traditional simulation and prototyping workflows through advanced optimization techniques. Applying industry-leading optimization techniques to your design process, lets you tackle the most pressing optimization challenges:

  • Single and multi-objective problems
  • Highly dimensional/multi-modal design space
  • Highly constrained (limited) objectives

The powerful optimization engine driving OASIS lets you find your best design option in less time than conventional optimization tools.

MAIN FUNCTION

User-Friendly Interface

OASIS provides an intuitive graphical user interface that enables users to efficiently define design objectives, constrain variables, and configure optimization parameters with streamlined setup procedures.

AI-Driven Optimization Engine

OASIS integrates four core optimization algorithms: single-objective global optimization, constrained single-objective optimization, multi-objective global optimization, and constrained multi-objective optimization. OASIS eliminates the need for algorithm selection or parameter tuning, allowing engineers to perform complex optimizations with one-click execution.

Integration Framework

OASIS provides three-tier interoperability solutions:

  • Script-Enabled Co-Simulation: Connects external tools through universal simulation interfaces using parametric control scripts.
  • CAE-Specific Plugin Integration: Delivers prebuilt plugins for mainstream CAE platforms to enable embedded optimization workflows.
  • API-Driven Optimization Services: Offers API endpoints for on-demand optimization service invocation and result streaming.

Visualization and Exploration

OASIS provides simple-to-use visualization and exploration tools, using banded 2D and 3D plots for effective multidimensional design variable analysis. All variable changes are recorded in the optimization results sheet, which automatically connects to visualization displays.

Decision Support

OASIS's decision support module enables users to efficiently analyze multi-objective problems and assign weighting criteria for in-depth evaluation of solution trade-offs.

Automated Reporting

Upon optimization completion, OASIS automatically generates structured technical reports for user analysis and presentation.
